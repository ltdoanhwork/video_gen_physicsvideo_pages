Video world models are powerful but computationally expensive. We benchmark training-free methods that accelerate inference while preserving generation quality for real-time robot control.

The Evaluation Gap

Standard video generation metrics miss what matters most for robotics applications.

What existing works measure

PSNR / SSIM
Pixel-level reconstruction — ignores semantic correctness
LPIPS
Perceptual similarity — still appearance-only
FID / FVD
Distribution-level quality — no per-sample physics check
IS (Inception Score)
Image diversity — irrelevant to action fidelity

What robotics actually needs

Physical Plausibility
Objects obey gravity, collisions, friction
Action Fidelity
Generated motion matches commanded actions
Task Completion
Can a policy succeed using simulated frames?
Temporal Consistency
No flickering, object permanence, smooth dynamics
Contact Accuracy
Grasp, push, lift events at correct timesteps

Key Insight

All 9 acceleration methods are training-free and plug-and-play, directly applicable to existing video world models without retraining. The challenge is maintaining physics-aware generation quality while achieving the 10+ FPS target for real-time robot control.
